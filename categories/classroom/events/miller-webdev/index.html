    Miller Solutions Development Inc. Industry Lecture: "Web Development: Latest Trends and Strategies"

    August 19, 2024
    ATA Hall
    1 pt

    What?

    On August 19, 2024, I attended an industry lecture by Miller Solutions Development Inc. at ATA Hall, specifically for the College of Computer Studies students. The event, which lasted from 2:00 PM to 5:00 PM, aimed to teach us about the latest trends in web development. I participated as a spectator along with my fellow CCS students.

    So What?

    Initially, I expected the lecture to be insightful, and it certainly met my expectations. I was particularly impressed by the discussion on the use of AI, which is not just a trend in the tech industry but is also becoming prevalent in other fields. One of the most memorable aspects of the lecture was the creative and humorous PowerPoint presentation. The presenters provided valuable insights into using GitHub Copilot for code suggestions, a ranking of popular JavaScript frameworks, and a demo of the Vue.js framework. Despite having no prior experience in web development, I found the lecture extremely useful.

    Now What?

    The experience taught me the proper use of AI in development, and I am now considering a subscription to GitHub Copilot for future projects. However, for the time being, using Language Server Protocols for code suggestions will suffice. As someone with a background in machine learning and game development, this activity was highly beneficial for expanding my knowledge into web development.

    How This Changed Me

    Because of this activity I am interested in web development and have begun learning HTML, CSS, and JavaScript. The insightful demonstrations provided during the lecture sparked this newfound interest.
